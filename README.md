# 🪐 Portfolio V2 — Alexon

Redesign completo do meu portfólio pessoal, migrado de HTML/CSS/JS puro para **React + TypeScript**, com foco em mobile-first, identidade visual espacial e animações mais ricas.

---

## ✨ O que tem até agora

- **Tela de loading** com nome animado estilo terminal, barra de progresso e estrelas piscando
- **Animação de transição** — o planeta voa da tela de loading direto para o header ao terminar
- **Navbar** responsiva com menu hamburguer (mobile) e links com underline animado (desktop)
- Paleta de cores e fontes definidas: `Press Start 2P` + `Share Tech Mono`

---

## 🛠 Stack

| Tecnologia | Uso |
|---|---|
| React 18 | Componentização e estado |
| TypeScript | Tipagem estática |
| Vite | Bundler e dev server |
| Styled Components | Estilização por componente |

---

## 🚀 Rodando localmente

```bash
# Instalar dependências
npm install

# Rodar em desenvolvimento
npm run dev

# Build de produção
npm run build
```
